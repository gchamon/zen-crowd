this will be a repo for zen-crowd, a browser extension for zen-browser aiming to bring two functionalities:

- subfolder background colorization, for a better visual separation of nested folders
- subtabs open in subfolders:
  - when opening a tab from another tab, the tab will be opened in a subfolder
  - clicking the folder header will open the original tab

Cloned in this repo are zen-browser desktop repo and zen-sidebery-mod for reference. They are excluded in the README.
